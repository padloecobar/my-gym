# Component Recipes (Short)

Use these recipes as a quick reference. The contract remains in `DESIGN_SYSTEM_CONTRACT.md`.

## Surface
`ui-card` / `ui-nav` / `ui-sheet__panel` use `data-surface="1|2|3"` for depth tiers.

## Pill / Chip
- `ui-pill` for metric/value pairs; `data-tone="strong"` for emphasis.
- `ui-chip` for toggles; `data-state="selected"` for active.

## Button
- `ui-button` with `data-variant` + `data-size`.
- `ui-icon-button` for square icon controls (48px min).

## Row
- `ui-row` for tap-able list rows; `data-density` controls padding.

## Sheet
- `ui-sheet` + `ui-sheet__panel|header|content|footer` compose dialogs.
